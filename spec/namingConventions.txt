Naming Conventions {
	
	Acronyms are treated as one word 
	N.A.S.A. -> Nasa
	J.S.O.N. -> Json
	S.I.M.D. -> Simd
	L.O.L. -> Lol
	
	One letter acronyms steal the uppercasing of the next letter.
	J. U.T.F.8 -> Jutf8
	U. Int -> Uint
	
	Use a module called "details" for implementation details that need to be exported.
	
	constant / immutable type variables/arguments use the same casing as normal types (HelloWorld)
	
	myVariable
	MY_CONSTANT_VARIABLE
	myJsonFunction
	myMacroFunction
	
	my_simd_module
	my_crate
	
	MyJsonType
	MyTraitLol
	primitive00
}
